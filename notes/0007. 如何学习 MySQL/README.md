## 1. 📝 概述

- `数据库基础 → SQL 语句 → 表操作 → 索引优化 → 事务与锁 → 存储过程与函数 → 性能调优 → 项目实战`
- 学习路径建议
  - 了解基础概念
    - 数据库、表、字段、SQL 语句基本语法
  - 掌握 SQL 查询
    - SELECT、INSERT、UPDATE、DELETE
    - JOIN 连接、子查询、聚合函数
  - 深入数据库设计
    - 数据范式、索引优化、视图、触发器、存储过程
  - 性能调优与高级特性
    - 查询优化、事务控制、锁机制、分区、集群
  - 实战项目练习
    - 构建博客系统、商城系统、论坛等
    - 接入后端语言（如 PHP、Java、Python、Node.js）
