## 1. 📝 概述

- 主键
  - 主键，又称主码，是表中一列或多列的组合。
  - 主键约束（Primary Key Constraint）要求主键列的数据唯一，并且不允许为空。
  - 主键能够唯一地标识表中的一条记录，可以结合外键来定义不同数据表之间的关系，并且可以加快数据库查询的速度。
  - 主键和记录之间的关系如同身份证和人之间的关系，它们之间是一一对应的。
- 主键分为两种类型：
  - 单字段主键和多字段联合主键。
